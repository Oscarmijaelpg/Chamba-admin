## 🚀 How to Use

### Start the Admin Dashboard
```bash
cd ChambaAdmin
npm install
npm run dev
```

Then open: `http://localhost:5173`

### Key Workflows

#### Approve a Withdrawal
1. Click "Finanzas" → "Retiros Pendientes"
2. Review user and amount
3. Click "Aprobar" or "Rechazar"
4. Add reason (optional)
5. Confirm

#### Handle a Report
1. Click "Reportes"
2. Click on pending report
3. Opens Moderation Modal
4. Review details
5. Choose action: Approve/Reject/Ban
6. System auto-notifies user

#### View Analytics
1. Click "Analytics"
2. See 30-day trends
3. User registrations chart
4. Revenue by day
5. Event distribution pie chart

#### Configure Alerts
1. Click "Alertas"
2. Set admin email
3. Toggle notification types
4. Click "Guardar Cambios"
5. Get email alerts based on preferences
